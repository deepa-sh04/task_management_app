# Tax Management App

A clean and responsive Tax Management web application built using React and TypeScript.  
Users can view tax records, filter them by country, and edit entries through a modal interface.

##  Live Demo
https://tax-management-app.vercel.app/

## Features
- Display tax records using **@tanstack/react-table**
- Edit tax name and country through a modal
- Filter by country with a searchable popup
- API integration for fetching and updating data
- Consistent and responsive UI

## Tech Stack
- React  
- TypeScript  
- @tanstack/react-table  
- CSS
